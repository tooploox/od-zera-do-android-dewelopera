Nowy projekt

  • File → New → Android Application Project
  • Package name: unikalny!
  • Minimum required SDK: 2.3 (to daje nam ponad 95% rynku!)
  • Target SDK: 4.3
  • Compile with: jak wyżej

Gratulacje! Bravo!
